> [!NOTE]
> #### For hexa<br />
> You need to input the entire byte. <br />
> Ex : ``00000001 = 1, 1 = 16777216``<br />
> #### For Long<br />
> It uses 2 ints encoding. You have to separator them with a coma or a space. <br />
> Ex : ``2,3917113 = 00000002003bc539, 23917113 = 016cf23900000000``
